# BMI Calculator GUI Project

This is a Python-based BMI Calculator application with a graphical user interface.

## Description
The application allows users to enter height and weight and calculates the Body Mass Index (BMI).
It also shows the BMI category such as Underweight, Normal, or Overweight.

## Technologies Used
- Python
- Tkinter (GUI)
- Basic Python Logic

## Features
- User-friendly GUI
- Accurate BMI calculation
- Instant result display

## How to Run
1. Install Python
2. Run the file using:
   python main.py
